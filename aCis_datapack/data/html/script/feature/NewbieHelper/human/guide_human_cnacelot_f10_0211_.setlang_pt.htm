L2Dsec Guia para Novatos:
Espadas de uma mão são as armas mais comuns. Suas habilidades são bem equilibradas em todos os aspectos. A Espada do Escudeiro com a qual você começou suas aventuras era uma delas. Se você não tem certeza de qual arma seria melhor para você, uma espada de uma mão é sempre uma escolha adequada.
Você pode usar a habilidade Golpe Poderoso com espadas de uma mão. Como você sabe, isso aumentará substancialmente seu dano. Uma tática básica para aqueles que usam a espada de uma mão é começar com um Golpe Poderoso e imediatamente seguir com um corte.
