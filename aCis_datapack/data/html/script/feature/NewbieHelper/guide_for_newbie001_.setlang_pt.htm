L2Dsec Guia para Novatos:
Um personagem novato é aquele que é novo no mundo de Lineage II. Certos benefícios estão disponíveis para personagens novatos até o nível 25.
Impostos e outras taxas levaram a um aumento acentuado nos preços de armaduras e armamentos. Para ajudar a compensar este problema, personagens novatos podem trocar seu equipamento usado por crédito total para a compra de novos equipamentos em qualquer loja de armas ou armaduras. Além disso, a ameaça de monstros pode ser esmagadora para personagens novatos. Eu os ajudo o máximo possível.
Venha me ver para magia de suporte que seja apropriada para o seu nível. Isso ajudará a construir a confiança de que você precisará para desafiar os monstros poderosos neste mundo.
