L2Dsec Guia para Novatos:
O Token de Viagem de Novato é reservado para jogadores de nível 20 ou inferior.
Ele permite que o jogador se mova para a cidade inicial de outra tribo através dos Guias de Novatos em cada cidade.
Tenha cuidado, pois em muitos casos, você pode aprender e praticar certas habilidades e missões que são exclusivas da cidade de cada tribo.
Uma vez que um jogador atinja o nível 20, ele não poderá mais se teletransportar usando o Token de Viagem de Novato.
